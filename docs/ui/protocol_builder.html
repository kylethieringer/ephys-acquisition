ui.protocol_builder

ProtocolBuilderDialog — non-modal dialog for composing and running trial protocols.

The dialog is created lazily by ~ui.main_window.MainWindow and kept alive (hidden, not destroyed) between uses so that in-progress protocol edits are preserved across open/close cycles.

Layout::

┌─────────────────────────────────────────────┐
│  Protocol Name  [___________________]        │
│  Clamp mode: ○ Current clamp  ○ Voltage clamp│
├────────────────────┬────────────────────────-┤
│  Stimuli           │  Edit selected stimulus  │
│  ┌──────────────┐  │  ─────────────────────  │
│  │ staircase 0  │  │  (stacked widget)       │
│  │ staircase 1  │  │                          │
│  └──────────────┘  │                          │
│  [+Add][−Remove]   │                          │
│  [↑Up ][↓Down  ]   │                          │
├────────────────────┴─────────────────────────┤
│  Global Timing                                │
│  Pre: [___]ms  Post: [___]ms  ITI: [___]ms   │
│  Repeats/stimulus: [___]                      │
│  CC only: Hyperpol amp [___]pA  dur [___]ms   │
│  VC only: AO scale [___]mV/V                  │
├───────────────────────────────────────────────┤
│  [Save Protocol]  [Load Protocol]             │
│  Estimated run time: XX min XX sec            │
│  [       Run Protocol       ]                 │
└───────────────────────────────────────────────┘

Developer notes

_stimuli is a Python list of ~acquisition.trial_protocol.StimulusDefinition objects kept in parallel with the QListWidget rows. The editor widgets operate on the currently selected row. _sync_editor_to_stim() writes the editor back to _stimuli before any read, ensuring consistency.

The dialog overrides closeEvent to hide itself rather than being destroyed, preserving all UI state between invocations.

 84class ProtocolBuilderDialog(QDialog):
 85    """Non-modal dialog for composing, saving, loading, and running trial protocols.
 86
 87    The dialog manages an ordered list of
 88    :class:`~acquisition.trial_protocol.StimulusDefinition` objects and
 89    global timing parameters.  When the user clicks "Run Protocol" the
 90    dialog serialises the protocol and emits ``protocol_run_requested``.
 91
 92    Signals:
 93        protocol_run_requested(dict): Emitted when "Run Protocol" is clicked.
 94            The dict contains the serialised protocol (same format as
 95            :func:`~acquisition.trial_protocol.protocol_to_dict`) plus a
 96            ``"save_dir"`` key with the current save directory path.
 97
 98    Attributes:
 99        _save_dir (str): Save directory kept in sync with
100            :class:`~ui.control_panel.ControlPanel` via :meth:`set_save_dir`.
101        _stimuli (list[StimulusDefinition]): Internal list of stimulus
102            objects, maintained in parallel with ``_stim_list`` (QListWidget).
103    """
104

441    def _on_add(self) -> None:
442        """Add a new default staircase stimulus entry to the list."""
443        stim = StimulusDefinition(
444            type="staircase",
445            name=f"Staircase {len(self._stimuli)}",
446        )
447        self._stimuli.append(stim)
448        item = QListWidgetItem(stim.name)
449        self._stim_list.addItem(item)
450        self._stim_list.setCurrentRow(len(self._stimuli) - 1)
451        self._update_estimated_time()
452
453    def _on_remove(self) -> None:
454        """Remove the currently selected stimulus from the list."""
455        row = self._stim_list.currentRow()
456        if row < 0:
457            return
458        self._stim_list.takeItem(row)
459        self._stimuli.pop(row)
460        self._update_estimated_time()
461
462    def _on_duplicate(self) -> None:
463        """Insert a deep copy of the selected stimulus after it in the list."""
464        row = self._stim_list.currentRow()
465        if row < 0:
466            return
467        self._sync_editor_to_stim()
468        import copy
469        stim = copy.deepcopy(self._stimuli[row])
470        stim.name = stim.name + " (copy)"
471        self._stimuli.insert(row + 1, stim)
472        item = QListWidgetItem(stim.name)
473        self._stim_list.insertItem(row + 1, item)
474        self._stim_list.setCurrentRow(row + 1)
475        self._update_estimated_time()
476
477    def _on_move_up(self) -> None:
478        """Move the selected stimulus one position up in the list."""
479        row = self._stim_list.currentRow()
480        if row <= 0:
481            return
482        self._swap_rows(row, row - 1)
483        self._stim_list.setCurrentRow(row - 1)
484
485    def _on_move_down(self) -> None:
486        """Move the selected stimulus one position down in the list."""
487        row = self._stim_list.currentRow()
488        if row < 0 or row >= len(self._stimuli) - 1:
489            return
490        self._swap_rows(row, row + 1)
491        self._stim_list.setCurrentRow(row + 1)
492
493    def _swap_rows(self, a: int, b: int) -> None:
494        """Swap two rows in both the internal list and the QListWidget.
495
496        Args:
497            a: Index of the first row.
498            b: Index of the second row.
499        """
500        self._stimuli[a], self._stimuli[b] = self._stimuli[b], self._stimuli[a]
501        item_a = self._stim_list.takeItem(a)
502        self._stim_list.insertItem(b, item_a)
503
504    def _on_row_changed(self, row: int) -> None:
505        """Populate the editor from the newly selected stimulus.
506
507        Args:
508            row: 0-based row index in the stimulus list.
509        """
510        if row < 0 or row >= len(self._stimuli):
511            return
512        self._populate_editor(self._stimuli[row])
513
514    def _on_stim_name_changed(self, text: str) -> None:
515        """Propagate an editor name change to the list widget and the internal list.
516
517        Args:
518            text: New stimulus name from the name QLineEdit.
519        """
520        row = self._stim_list.currentRow()
521        if row < 0 or row >= len(self._stimuli):
522            return
523        self._stimuli[row].name = text
524        self._stim_list.item(row).setText(text)
525
526    def _on_stim_type_changed(self) -> None:
527        """Switch the stacked editor page when the stimulus type radio button changes."""
528        row = self._stim_list.currentRow()
529        if row < 0:
530            return
531        new_type = "staircase" if self._sc_type_rb.isChecked() else "voltage_step"
532        self._stimuli[row].type = new_type
533        self._stim_stack.setCurrentIndex(0 if new_type == "staircase" else 1)
534        self._update_estimated_time()

540    def _populate_editor(self, stim: StimulusDefinition) -> None:
541        """Fill editor widgets from a :class:`~acquisition.trial_protocol.StimulusDefinition`.
542
543        Blocks signals during fill to avoid spurious ``_update_estimated_time``
544        calls.
545
546        Args:
547            stim: Stimulus whose values should populate the editor.
548        """
549        self._stim_name_edit.blockSignals(True)
550        self._stim_name_edit.setText(stim.name)
551        self._stim_name_edit.blockSignals(False)
552
553        is_sc = stim.type == "staircase"
554        self._sc_type_rb.blockSignals(True)
555        self._vc_type_rb.blockSignals(True)
556        self._sc_type_rb.setChecked(is_sc)
557        self._vc_type_rb.setChecked(not is_sc)
558        self._sc_type_rb.blockSignals(False)
559        self._vc_type_rb.blockSignals(False)
560        self._stim_stack.setCurrentIndex(0 if is_sc else 1)

562        if is_sc:
563            self._sc_min.setValue(stim.min_pA or 0.0)
564            self._sc_max.setValue(stim.max_pA or 400.0)
565            self._sc_step.setValue(stim.step_pA or 100.0)
566            self._sc_width.setValue(stim.step_width_ms or 500.0)
567            self._sc_gap.setValue(stim.gap_ms or 100.0)
568            self._sc_reps.setValue(stim.staircase_repeats or 1)
569        else:
570            self._vs_step_mv.setValue(stim.step_mV or -40.0)
571            self._vs_duration.setValue(stim.duration_ms or 500.0)

573    def _sync_editor_to_stim(self) -> None:
574        """Write editor widget values back into the selected StimulusDefinition.

577        displayed edits are captured.  No-op if no row is selected.
578        """
579        row = self._stim_list.currentRow()

581            return

583        stim.name = self._stim_name_edit.text()
584        stim.type = "staircase" if self._sc_type_rb.isChecked() else "voltage_step"
585
586        if stim.type == "staircase":
587            stim.min_pA            = self._sc_min.value()
588            stim.max_pA            = self._sc_max.value()
589            stim.step_pA           = self._sc_step.value()
590            stim.step_width_ms     = self._sc_width.value()
591            stim.gap_ms            = self._sc_gap.value()
592            stim.staircase_repeats = self._sc_reps.value()
593        else:
594            stim.step_mV     = self._vs_step_mv.value()
595            stim.duration_ms = self._vs_duration.value()

602        """Construct a :class:`~acquisition.trial_protocol.TrialProtocol` from all UI fields.

463    def _on_duplicate(self) -> None:
464        """Insert a deep copy of the selected stimulus after it in the list."""
465        row = self._stim_list.currentRow()
466        if row < 0:
467            return
468        self._sync_editor_to_stim()
469        import copy
470        stim = copy.deepcopy(self._stimuli[row])
471        stim.name = stim.name + " (copy)"
472        self._stimuli.insert(row + 1, stim)
473        item = QListWidgetItem(stim.name)
474        self._stim_list.insertItem(row + 1, item)
475        self._stim_list.setCurrentRow(row + 1)
476        self._update_estimated_time()

478    def _on_move_up(self) -> None:
479        """Move the selected stimulus one position up in the list."""
480        row = self._stim_list.currentRow()
481        if row <= 0:
482            return
483        self._swap_rows(row, row - 1)
484        self._stim_list.setCurrentRow(row - 1)

487        """Move the selected stimulus one position down in the list."""
488        row = self._stim_list.currentRow()
489        if row < 0 or row >= len(self._stimuli) - 1:

541    def _populate_editor(self, stim: StimulusDefinition) -> None:
542        """Fill editor widgets from a :class:`~acquisition.trial_protocol.StimulusDefinition`.

547        Args:

565            self._sc_max.setValue(stim.max_pA or 400.0)
566            self._sc_step.setValue(stim.step_pA or 100.0)
567            self._sc_width.setValue(stim.step_width_ms or 500.0)
568            self._sc_gap.setValue(stim.gap_ms or 100.0)
569            self._sc_reps.setValue(stim.staircase_repeats or 1)
570        else:
571            self._vs_step_mv.setValue(stim.step_mV or -40.0)
572            self._vs_duration.setValue(stim.duration_ms or 500.0)

Non-modal dialog for composing, saving, loading, and running trial protocols.

The dialog manages an ordered list of ~acquisition.trial_protocol.StimulusDefinition objects and global timing parameters. When the user clicks "Run Protocol" the dialog serialises the protocol and emits protocol_run_requested.

Signals:

protocol_run_requested(dict): Emitted when "Run Protocol" is clicked. The dict contains the serialised protocol (same format as ~acquisition.trial_protocol.protocol_to_dict()) plus a "save_dir" key with the current save directory path.

Attributes:
  • _save_dir (str): Save directory kept in sync with ~ui.control_panel.ControlPanel via set_save_dir().
  • _stimuli (list[StimulusDefinition]): Internal list of stimulus objects, maintained in parallel with _stim_list (QListWidget).
