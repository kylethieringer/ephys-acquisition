acquisition.trial_mode

TrialAcquisition — orchestrates DAQWorker, CameraWorker, and TrialSaver for trial-based acquisition.

Lifecycle::

start()            → DAQ + camera workers start (same as ContinuousAcquisition)
run_protocol(...)  → builds trial order, enters state machine
cancel_protocol()  → graceful cancel after the current trial finishes
stop()             → cancel any active run, then shut down workers

State machine

The state machine advances by counting AI samples. It runs entirely in the GUI thread via the _on_ai_chunk Qt slot (connected with AutoConnection from DAQWorker's data_ready signal), so no locking is needed on state variables.

States and transitions::

_S_IDLE
    │  run_protocol() called
    ▼
_S_ITI  (inter-trial interval — AO silent, TTL off)
    │  iti_samples counted
    ▼
_S_PRE  (pre-baseline — TTL fires, data accumulates)
    │  pre_samples counted
    ▼
_S_TRIAL  (stimulus + post — data accumulates)
    │  (total_samples − pre_samples) counted
    │  → TTL stops, trial saved
    ▼
_S_ITI  ← repeat until all trials done, or cancel requested
    │  all trials done / cancel
    ▼
_S_DONE

Camera TTL is active for the full trial window: pre + stim + post. The AO waveform is pre-loaded during the ITI so the task rebuild (~5–10 ms) completes well before any current is output.

Developer notes

All public methods are called from the GUI thread. Data accumulation and state transitions happen in _on_ai_chunk (GUI thread), so no locking is required for state variables.

272    def start(self) -> None:
273        """Start the DAQ and camera workers.
274
275        Mirrors :meth:`~acquisition.continuous_mode.ContinuousAcquisition.start`.
276        No-op if already running.
277        """
278        if self._is_running:
279            return
280
281        self._daq_worker = DAQWorker(self._frame_rate_hz, self._exposure_ms)
282        self._daq_worker.data_ready.connect(self._on_ai_chunk)
283        self._daq_worker.error_occurred.connect(self._handle_error)
284        self._daq_worker.start()
285
286        self._camera_worker = CameraWorker(self._exposure_ms)
287        self._camera_worker.frame_ready.connect(self._on_camera_frame)
288        self._camera_worker.error_occurred.connect(self._handle_camera_error)
289        self._camera_worker.start()
290
291        self._is_running = True
292        self.started.emit()
293
294    def stop(self) -> None:
295        """Cancel any running protocol and stop all hardware workers.
296
297        Force-closes the HDF5 file if open, ensures the TTL is off, then
298        tears down the camera and DAQ worker threads.  No-op if not running.
299        """
300        if not self._is_running:
301            return
302
303        if self._video_writer is not None:
304            self._video_writer.release()
305            self._video_writer = None
306
307        self._finalise_trial_metadata_json(self._trial_pos)
308
309        if self._saver.is_open:
310            self._saver.close()
311        self._state = _S_IDLE
312
313        if self._daq_worker is not None:
314            self._daq_worker.stop_ttl()
315            self._daq_worker.clear_stimulus_waveform()
316
317        self._teardown_camera()
318
319        if self._daq_worker is not None:
320            self._daq_worker.stop()
321            self._daq_worker.wait(5000)
322            self._daq_worker = None
323
324        self._is_running = False
325        self.stopped.emit()

331    def run_protocol(
332        self,
333        protocol: TrialProtocol,
334        save_dir: str | Path,
335        subject_metadata: dict,
336    ) -> None:
337        """Start a protocol run.
338
339        Creates the HDF5 file, builds the shuffled trial order, and enters
340        the state machine at the ITI state.
341
342        Workers must be started first via :meth:`start`.  No-op if not
343        running or if ``protocol.stimuli`` is empty.
344
345        Args:
346            protocol: :class:`~acquisition.trial_protocol.TrialProtocol`
347                defining stimuli, timing, and clamp mode.
348            save_dir: Root directory for saving the HDF5 file.
349            subject_metadata: Subject information dict (same keys as
350                :meth:`~acquisition.data_saver.ContinuousSaver.open`).
351        """
352        if not self._is_running or not protocol.stimuli:
353            return
354
355        self._protocol         = protocol
356        self._subject_metadata = subject_metadata
357        self._trial_order      = build_trial_order(protocol)
358        self._cancel_requested = False
359        self._trial_pos        = 0
360
361        channel_defs = (
362            AI_CHANNELS_VC
363            if protocol.clamp_mode == "voltage_clamp"
364            else AI_CHANNELS
365        )
366
367        self._saver.open(
368            save_dir          = save_dir,
369            protocol          = protocol,
370            trial_order       = self._trial_order,
371            subject_metadata  = subject_metadata,
372            channel_defs      = channel_defs,
373        )
374        self._video_folder = self._saver.folder
375        self._write_trial_metadata_json(protocol, channel_defs)
376
377        self._iti_samples = _ms_to_samples(protocol.iti_ms)
378        self._enter_iti()
379
380    def cancel_protocol(self) -> None:
381        """Request a graceful cancel of the running protocol.
382
383        The current trial (if any) finishes saving before the run stops.
384        No data from completed trials is lost.  ``protocol_cancelled`` is
385        emitted with the count of completed trials.
386        """
387        self._cancel_requested = True

393    def _enter_iti(self) -> None:
394        """Enter the ITI state: AO silent, TTL off, counting ITI samples.
395
396        Does not issue any hardware commands — the TTL and AO are already
397        off from the previous trial end (or from startup).
398        """
399        self._state          = _S_ITI
400        self._sample_counter = 0
401
402    def _enter_pre(self) -> None:
403        """Enter the PRE state: load waveform, fire TTL, start accumulating data.
404
405        Sequence:
406
407        1. Look up the next stimulus in ``_trial_order``.
408        2. Build the full AO waveform (pre zeros + hyperpol + staircase + post zeros).
409        3. Pre-allocate the trial data buffer.
410        4. Load the waveform into the AO task (rebuilds the nidaqmx task).
411        5. Fire the camera TTL.
412        6. Reset the sample counter and emit ``trial_started``.
413
414        The waveform is loaded *before* the TTL fires so the nidaqmx task
415        rebuild (~5–10 ms) completes before the camera starts capturing.
416        """
417        stim_idx = self._trial_order[self._trial_pos]
418        stim_def = self._protocol.stimuli[stim_idx]
419
420        waveform = build_trial_waveform(stim_def, self._protocol)
421        self._total_samples = len(waveform)
422        self._pre_samples   = _ms_to_samples(self._protocol.pre_ms)
423
424        self._trial_buf = np.empty(
425            (N_AI_CHANNELS, self._total_samples), dtype=np.float64
426        )
427        self._buf_ptr = 0
428
429        # Prepare video writer for this trial (opened lazily on first frame)
430        self._video_writer = None
431        if self._video_folder is not None and self._saver.path is not None:
432            self._video_path = self._video_folder / f"{self._saver.path.stem}_{self._trial_pos + 1:03d}.avi"
433        else:
434            self._video_path = None
435
436        if self._daq_worker is not None:
437            # Baseline trials keep AO at 0 V — no waveform needed (AO is already
438            # silent from the previous trial's clear_stimulus_waveform call, or
439            # from the initial zero-output at DAQ startup).  Rebuilding the AO
440            # task with a zeros array is unnecessary and can interfere with the
441            # TTL counter start, so we skip it for baseline.
442            if stim_def.type != "baseline":
443                self._daq_worker.set_stimulus_waveform(waveform.reshape(1, -1))
444            self._daq_worker.start_ttl()
445
446        self._state          = _S_PRE
447        self._sample_counter = 0
448
449        total = len(self._trial_order)
450        self.trial_started.emit(self._trial_pos, total)
451        self.progress_updated.emit(self._trial_pos, total)
452
453    def _on_trial_window_start(self) -> None:
454        """Transition from PRE to TRIAL state (no hardware change).
455
456        Called when the pre-baseline sample threshold is reached.
457        Resets the counter so ``_check_state_transition`` can measure the
458        remaining (stim + post) samples.
459        """
460        self._state          = _S_TRIAL
461        self._sample_counter = 0
462
463    def _on_trial_end(self) -> None:
464        """Handle the end of a trial: stop TTL, save data, advance the state machine.
465
466        Sequence:
467
468        1. Stop the camera TTL.
469        2. Clear the AO waveform (revert ao0 to zeros).
470        3. Pre-create the HDF5 group (``begin_trial``).
471        4. Write the accumulated buffer (``write_trial``).
472        5. Emit ``trial_finished``.
473        6. Increment ``_trial_pos`` and call :meth:`_advance`.
474        """
475        if self._daq_worker is not None:
476            self._daq_worker.stop_ttl()
477            self._daq_worker.clear_stimulus_waveform()
478
479        # Close the video writer for this trial before saving HDF5 metadata
480        if self._video_writer is not None:
481            self._video_writer.release()
482            self._video_writer = None
483
484        stim_idx = self._trial_order[self._trial_pos]
485        stim_def = self._protocol.stimuli[stim_idx]
486        onset_iso = datetime.datetime.now().isoformat()
487
488        video_filename = self._video_path.name if self._video_path is not None else ""
489        if self._trial_buf is not None:
490            self._saver.write_trial(
491                trial_index    = self._trial_pos,
492                stimulus_index = stim_idx,
493                stimulus_name  = stim_def.name,
494                onset_time     = onset_iso,
495                data           = self._trial_buf,
496                video_filename = video_filename,
497            )
498
499        total = len(self._trial_order)
500        self.trial_finished.emit(self._trial_pos, total)
501        self._trial_pos += 1
502
503        self._advance()
504
505    def _advance(self) -> None:
506        """Move to the next trial or finish the run.
507
508        If a cancel was requested or all trials are done, closes the HDF5
509        file and emits the appropriate completion signal.  Otherwise enters
510        the next ITI.
511        """
512        total = len(self._trial_order)
513
514        if self._cancel_requested or self._trial_pos >= total:
515            self._state = _S_DONE
516            self._finalise_trial_metadata_json(self._trial_pos)
517            path = self._saver.close()
518            if self._cancel_requested:
519                self.protocol_cancelled.emit(self._trial_pos)
520            else:
521                if path is not None:
522                    self.protocol_finished.emit(path)
523                else:
524                    self.protocol_finished.emit(Path())
525        else:
526            self._enter_iti()

532    def _on_ai_chunk(self, chunk: NDArray[np.float64]) -> None:
533        """Receive an AI data chunk and drive the state machine (~100 Hz).
534
535        Forwards the chunk to the display callback unconditionally, then
536        accumulates data into the trial buffer if in PRE or TRIAL state,
537        and calls :meth:`_check_state_transition` to advance the state.
538
539        Args:
540            chunk: ``(N_AI_CHANNELS, CHUNK_SIZE)`` float64 array in Volts.
541        """
542        if self._on_new_data is not None:
543            self._on_new_data(chunk)
544
545        if self._state == _S_IDLE or self._state == _S_DONE:
546            return
547
548        n = chunk.shape[1]
549
550        if self._state in (_S_PRE, _S_TRIAL):
551            if self._trial_buf is not None:
552                space = self._total_samples - self._buf_ptr
553                take  = min(n, space)
554                if take > 0:
555                    self._trial_buf[:, self._buf_ptr : self._buf_ptr + take] = chunk[:, :take]
556                    self._buf_ptr += take
557
558        self._sample_counter += n
559        self._check_state_transition()
560
561    def _check_state_transition(self) -> None:
562        """Evaluate whether a state boundary has been crossed.
563
564        Checks if the current sample counter has reached the threshold for
565        the current state and calls the appropriate transition method.
566        The counter is decremented by the threshold (not zeroed) to preserve
567        any overshoot for the next state.
568        """
569        if self._state == _S_ITI:
570            if self._sample_counter >= self._iti_samples:
571                self._sample_counter -= self._iti_samples
572                self._enter_pre()
573
574        elif self._state == _S_PRE:
575            if self._sample_counter >= self._pre_samples:
576                self._sample_counter -= self._pre_samples
577                self._on_trial_window_start()
578
579        elif self._state == _S_TRIAL:
580            stim_post = self._total_samples - self._pre_samples
581            if self._sample_counter >= stim_post:
582                self._sample_counter -= stim_post
583                self._on_trial_end()

604    def _write_video_frame(self, frame: NDArray) -> None:
605        """Lazily open a cv2.VideoWriter on the first frame of a trial, then write.
606
607        ``uint16`` frames are downsampled to ``uint8`` by right-shifting 8 bits.
608        Colour frames are converted from RGB (Basler) to BGR (OpenCV).
609
610        Args:
611            frame: Camera frame array from the CameraWorker.
612        """
613        if not HAS_CV2 or self._video_path is None:
614            return
615
616        if self._video_writer is None:
617            h, w = frame.shape[:2]
618            is_color = frame.ndim == 3
619            fps = get_actual_frame_rate(self._frame_rate_hz)

621            self._video_writer = cv2.VideoWriter(

403    def _enter_pre(self) -> None:
404        """Enter the PRE state: load waveform, fire TTL, start accumulating data.
405
406        Sequence:
407
408        1. Look up the next stimulus in ``_trial_order``.
409        2. Build the full AO waveform (pre zeros + hyperpol + staircase + post zeros).
410        3. Pre-allocate the trial data buffer.
411        4. Load the waveform into the AO task (rebuilds the nidaqmx task).
412        5. Fire the camera TTL.
413        6. Reset the sample counter and emit ``trial_started``.
414
415        The waveform is loaded *before* the TTL fires so the nidaqmx task
416        rebuild (~5–10 ms) completes before the camera starts capturing.
417        """
418        stim_idx = self._trial_order[self._trial_pos]
419        stim_def = self._protocol.stimuli[stim_idx]
420
421        waveform = build_trial_waveform(stim_def, self._protocol)
422        self._total_samples = len(waveform)
423        self._pre_samples   = _ms_to_samples(self._protocol.pre_ms)
424
425        self._trial_buf = np.empty(
426            (N_AI_CHANNELS, self._total_samples), dtype=np.float64
427        )
428        self._buf_ptr = 0
429
430        # Prepare video writer for this trial (opened lazily on first frame)
431        self._video_writer = None
432        if self._video_folder is not None and self._saver.path is not None:
433            self._video_path = self._video_folder / f"{self._saver.path.stem}_{self._trial_pos + 1:03d}.avi"
434        else:
435            self._video_path = None
436
437        if self._daq_worker is not None:
438            # Baseline trials keep AO at 0 V — no waveform needed (AO is already
439            # silent from the previous trial's clear_stimulus_waveform call, or
440            # from the initial zero-output at DAQ startup).  Rebuilding the AO
441            # task with a zeros array is unnecessary and can interfere with the
442            # TTL counter start, so we skip it for baseline.
443            if stim_def.type != "baseline":
444                self._daq_worker.set_stimulus_waveform(waveform.reshape(1, -1))
445            self._daq_worker.start_ttl()
446
447        self._state          = _S_PRE
448        self._sample_counter = 0
449
450        total = len(self._trial_order)
451        self.trial_started.emit(self._trial_pos, total)
452        self.progress_updated.emit(self._trial_pos, total)

467        Sequence:

469        1. Stop the camera TTL.

471        3. Pre-create the HDF5 group (``begin_trial``).
472        4. Write the accumulated buffer (``write_trial``).

474        6. Increment ``_trial_pos`` and call :meth:`_advance`.

510        file and emits the appropriate completion signal.  Otherwise enters
511        the next ITI.

513        total = len(self._trial_order)

533    def _on_ai_chunk(self, chunk: NDArray[np.float64]) -> None:
534        """Receive an AI data chunk and drive the state machine (~100 Hz).
535
536        Forwards the chunk to the display callback unconditionally, then
537        accumulates data into the trial buffer if in PRE or TRIAL state,
538        and calls :meth:`_check_state_transition` to advance the state.
539
540        Args:
541            chunk: ``(N_AI_CHANNELS, CHUNK_SIZE)`` float64 array in Volts.
542        """
543        if self._on_new_data is not None:
544            self._on_new_data(chunk)
545
546        if self._state == _S_IDLE or self._state == _S_DONE:
547            return
548
549        n = chunk.shape[1]
550
551        if self._state in (_S_PRE, _S_TRIAL):
552            if self._trial_buf is not None:
553                space = self._total_samples - self._buf_ptr
554                take  = min(n, space)
555                if take > 0:
556                    self._trial_buf[:, self._buf_ptr : self._buf_ptr + take] = chunk[:, :take]
557                    self._buf_ptr += take
558
559        self._sample_counter += n
560        self._check_state_transition()
561
562    def _check_state_transition(self) -> None:
563        """Evaluate whether a state boundary has been crossed.
564
565        Checks if the current sample counter has reached the threshold for
566        the current state and calls the appropriate transition method.
567        The counter is decremented by the threshold (not zeroed) to preserve
568        any overshoot for the next state.
569        """
570        if self._state == _S_ITI:
571            if self._sample_counter >= self._iti_samples:
572                self._sample_counter -= self._iti_samples
573                self._enter_pre()
574
575        elif self._state == _S_PRE:
576            if self._sample_counter >= self._pre_samples:
577                self._sample_counter -= self._pre_samples
578                self._on_trial_window_start()
579
580        elif self._state == _S_TRIAL:
581            stim_post = self._total_samples - self._pre_samples
582            if self._sample_counter >= stim_post:
583                self._sample_counter -= stim_post
584                self._on_trial_end()
