Quick Start

Load & plot a recording

Use utils.data_loader to inspect any saved HDF5 file without launching the GUI. 

from utils.data_loader import load_hdf5, plot_data

# Load raw data into a dict
d = load_hdf5("recording_2025-06-01.h5")
print(d['channel_names'])   # ['Im (pA)', 'Vm (mV)', ...]
print(d['data'].shape)       # (n_channels, n_samples)

# Plot first 10 seconds, one subplot per channel
plot_data("recording_2025-06-01.h5", time_range=(0, 10))

Quality Control

Re-run QC or run alignment

QC runs automatically after every save. To re-run it on an existing file, or to run the standalone hardware-alignment check, use these CLI entry points. 

# Re-run the full post-recording QC pipeline on an existing file
from analysis.qc.report import run_qc
result = run_qc("recording_2025-06-01.h5")
print(result['status'])      # 'pass', 'warn', or 'fail'
print(result['html_path'])   # path to the generated HTML report

# Standalone hardware alignment (run with Axon Patch-1U model cell)
# python -m analysis.qc_alignment --save-dir D:/data
# python -m analysis.qc_alignment --save-dir D:/data --no-phase-b

Concepts

Domain & design notes

Refer to config for all physical-unit definitions and channel scaling.

Current Clamp (CC)
Inject current in pA; record membrane voltage in mV. AO scale: 400 pA/V — write X/400 V to ao0 to inject X pA.
Voltage Clamp (VC)
Command voltage in mV; record clamping current in pA. AO scale: 20 mV/V — write X/20 V for X mV command.
TTL Synchronization
5 V pulses via counter task on PFI12. Triggers Basler camera on Line1 — phase-locked to acquisition.
Trial State Machine
IDLE → ITI → PRE → TRIAL → DONE. Staircase amplitudes step automatically across the protocol.
Clock Topology
AI task = master clock. AO slaved via ai/SampleClock. CTR runs from the 100 MHz on-board timebase.
HDF5 Layout
/data/analog_input — float64 (n_ch × n_samp), LZF compressed. Full metadata in /metadata group.
Post-recording QC
Automatic background QC after every save: sample-count consistency, finite values, stimulus events, TTL ↔ video frame-count drift, and an interactive plotly channel overview.
Hardware Alignment
Weekly rig check via python -m analysis.qc_alignment. Phase A (no model cell): loopback latency, crosstalk, TTL jitter. Phase B (Axon Patch-1U, 500 MΩ): CC/VC scaling, noise, τ.
